Visual Studio Code

Visual Studio Code, or VSC, is an offline IDE, that is free to download.You can download it by going to https://code.visualstudio.com/download.

Console Screen

The Console Screen is a window that you can open, and pass messages to. This can be used to check the status of the code. or the recheck an output before showing it to the user. The console screen also shows any errors in the code while it is running. There are many more functions of the Console Screen.

You can open the Console Screen by opening the "Inspect" window, and opening the "Console" tab. You can open the Inspect menu bu right-clicking and selecting "Inspect" or pressing Ctrl+Shift+i (Windows) or Command+Option+i (Mac).
